TRADEBASE FEATURE UPDATE
========================

This update includes two major UI improvements:

1) Replace the Leads section with a Clients section
2) Add a persistent Quick Create floating button across the app

-------------------------------------

CLIENTS TAB (REPLACES LEADS)
=============================

Purpose
Clients should become the central record for every person the contractor works with. 
All activity related to a client should be visible in one place: quotes, jobs, invoices, notes, photos, and AI results.

Navigation Update

Bottom navigation should now be:

Home
Clients
Quotes
Jobs
Money
More

Remove "Leads" from the main navigation.

-------------------------------------

-------------------------------------

FLOATING QUICK CREATE BUTTON
=============================

Add a persistent floating button across the app.

Location:
Bottom-right corner above the navigation bar.

Label:

+ Create

Purpose:
Allow users to create records instantly from any screen without switching tabs.

When tapped, open an action menu with these options:

New Client
New Quote
New Job
New Invoice
Add Note
Upload Photo
AI Job Capture

Each option should immediately open the appropriate creation flow.

Design requirements:

Large mobile-friendly tap target
Always visible on main screens
Does not block important content
Works on Home, Clients, Quotes, Jobs, and Money pages
